# MeiYouAd-Downloader
程序下载：https://github.com/nilaoda/MeiYouAd-Downloader/releases

## 目前支持的片库
M3U8：芒果、优酷、爱奇艺、乐视

分段：腾讯、搜狐

## M3U8片库怎么没反应
在 ``IDM`` 的 ``选项-常规`` 中将本程序的可执行文件（.exe）设置为被嗅探浏览器之一。

然后再次解析，程序左下角将出现 ``从该页面下载视频`` 悬浮框，此时即可使用idm完成后续操作

## 可以记住账号密码吗
不可以。但是你可以在``sets.ini``中写入您的账号密码，软件再次开启时会尝试读取。
